# termux-services

## Start a service

```bash
sv up sshd
```

```
## stop  a service 
```bash
sv down sshd
```

```
```

## restart  a service

```bash
sv restart sshd
```

## check status of  a service

```bash
sv status sshd
```

## enable a service

```bash
sv-enable sshd
```

## disable a service

```bash
sv-disable sshd
```
