# rofi-matugen — Material You Rofi Themes

A cohesive set of Rofi themes built on the **Material Design 3 color system**, fully integrated with [matugen](https://github.com/InioX/matugen) for automatic wallpaper-driven color generation. Every surface, accent, and state follows the M3 color token spec, so your entire launcher suite updates the moment your wallpaper changes.

---

## File Overview

```
~/.config/rofi/
├── colors.rasi                  ← Generated by matugen (or use the fallback)
├── shared.rasi                  ← Common reset / scrollbar / message styles
├── launcher.rasi                ← App launcher   (drun mode)
├── runner.rasi                  ← Command runner  (run mode)
├── window.rasi                  ← Window switcher (window mode)
├── clipboard.rasi               ← Clipboard history (cliphist / greenclip)
├── powermenu.rasi               ← Power / session menu
├── wallpaper.rasi               ← Wallpaper switcher (grid thumbnail)
├── emoji.rasi                   ← Emoji picker (rofimoji)
├── ssh.rasi                     ← SSH host picker
├── screenshot.rasi              ← Screenshot mode picker
├── matugen/
│   └── rofi-colors.rasi         ← Matugen template (copy to matugen config dir)
└── scripts/
    ├── powermenu.sh
    ├── clipboard.sh
    ├── wallpaper.sh
    ├── screenshot.sh
    └── emoji.sh
```

## Installation

### 1. Copy theme files

```bash
# Backup existing config
cp -r ~/.config/rofi ~/.config/rofi.bak

# Install themes
cp -r rofi-matugen-theme/* ~/.config/rofi/
chmod +x ~/.config/rofi/scripts/*.sh
```

### 2. Install dependencies

**Fonts (required)**
```bash
# Arch
yay -S ttf-jetbrains-mono-nerd noto-fonts-emoji

# Debian / Ubuntu
sudo apt install fonts-nerd-fonts-jetbrains-mono fonts-noto-color-emoji
```

**Icons (required)**
```bash
# Arch
sudo pacman -S papirus-icon-theme

# Debian / Ubuntu
sudo apt install papirus-icon-theme
```

**Clipboard (Wayland)**
```bash
sudo pacman -S cliphist wl-clipboard
# Add to Hyprland:
# exec-once = wl-paste --type text  --watch cliphist store
# exec-once = wl-paste --type image --watch cliphist store
```

**Wallpaper (Wayland)**
```bash
sudo pacman -S swww
# exec-once = swww-daemon
```

## Matugen Integration

### Setup

```bash
# Install matugen
cargo install matugen
# or: yay -S matugen-bin
```

**`~/.config/matugen/config.toml`** — add the Rofi template:

```toml
[config]
reload_apps = true

[config.reload_apps_list]
waybar = true
dunst  = true

[[config.templates]]
input_path  = "~/.config/rofi/matugen/rofi-colors.rasi"
output_path = "~/.config/rofi/colors.rasi"
```

### Regenerate colors on wallpaper change

```bash
matugen image /path/to/wallpaper.jpg
```

The wallpaper script (`scripts/wallpaper.sh`) calls `matugen` automatically after applying a new wallpaper. Every Rofi theme picks up the new `colors.rasi` on next launch — no restart required.

## Keybinding Reference

Add these to your compositor config:

### Hyprland (`~/.config/hypr/keybinds.conf`)

```ini
$mod = SUPER

# App launcher
bind = $mod,       Space,    exec, rofi -show drun -theme ~/.config/rofi/launcher.rasi

# Window switcher
bind = $mod,       Tab,      exec, rofi -show window -theme ~/.config/rofi/window.rasi

# Run dialog
bind = $mod SHIFT, Space,    exec, rofi -show run -theme ~/.config/rofi/runner.rasi

# SSH
bind = $mod,       S,        exec, rofi -show ssh -theme ~/.config/rofi/ssh.rasi

# Clipboard
bind = $mod,       V,        exec, ~/.config/rofi/scripts/clipboard.sh

# Power menu
bind = $mod,       Escape,   exec, ~/.config/rofi/scripts/powermenu.sh

# Wallpaper picker
bind = $mod SHIFT, W,        exec, ~/.config/rofi/scripts/wallpaper.sh

# Screenshot menu
bind = ,           Print,    exec, ~/.config/rofi/scripts/screenshot.sh

# Emoji picker
bind = $mod SHIFT, E,        exec, ~/.config/rofi/scripts/emoji.sh
```

### Sway (`~/.config/sway/config`)

```
set $mod Mod4

bindsym $mod+space       exec rofi -show drun -theme ~/.config/rofi/launcher.rasi
bindsym $mod+Tab         exec rofi -show window -theme ~/.config/rofi/window.rasi
bindsym $mod+Shift+space exec rofi -show run -theme ~/.config/rofi/runner.rasi

bindsym $mod+Escape      exec ~/.config/rofi/scripts/powermenu.sh
```

## Color System

Each theme uses **semantic color aliases** from `colors.rasi`, never raw hex values. This means:

| Role | Token | Usage |
|------|-------|-------|
| Primary accent | `@primary` | Prompt icons, selected highlights |
| Selected bg | `@primary-container` | Launcher selected item |
| Secondary accent | `@secondary` | Window switcher, wallpaper switcher |
| Tertiary accent | `@tertiary` | Clipboard, runner, SSH |
| Destructive | `@error` | Power menu shutdown/reboot |
| Base surface | `@surface-container` | Window background |
| Elevated surface | `@surface-container-high` | Input boxes, alt items |
| Borders | `@outline-variant` | Window border, separators |
| Body text | `@on-surface` | Primary text |
| Muted text | `@on-surface-variant` | Placeholder, hints |

## Customisation Tips

**Change border radius** — edit each `.rasi` and adjust `border-radius` on `window {}`.

**Hide window border** — set `border: 0px;` in the `window {}` block.

**Blur** — if your compositor supports blur for rofi windows, set `transparency: "real"` (already done) and add a blur rule:

```ini
# Hyprland
layerrule = blur, rofi
layerrule = ignorezero, rofi
```

**Different font** — change the `font:` in `shared.rasi` and it propagates to all themes:

```rasi
* { font: "Iosevka Nerd Font 11"; }
```

**Light mode** — swap `colors.rasi` to a light Material You palette (matugen supports `--type light`):

```bash
matugen image /path/to/wallpaper.jpg --type light
```
